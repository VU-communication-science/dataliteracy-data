## Data Literacy data

This repo contains the data used in the VU Data Literacy book. These files can be accessed via `https://data.dataliteracy.cc/[filesname]`.

You can also make subfolders. If you create the file `KWOM/assignment_1.csv`, you can access it as `https://data.dataliteracy.cc/KWOM/assignment_1.csv`.

Note that you do not have to clone the repo to add or change files. You can do this directly via the Github website.
