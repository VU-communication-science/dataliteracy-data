## Data Literacy data

This repo contains the data used in the VU Data Literacy book. These files can be accessed via `https://data.dataliteracy.cc` using the same folder and filename. For instance, the file `simulated/practice_data.csv` can be download from `https://data.dataliteracy.cc/simulated/practice_data.csv`.

Note that you do not have to clone the repo to add or change files. You can do this directly via the Github website.
