Data Literacy Data

This site hosts the datasets used in the VU Data Literacy book. Every file in the repository is accessible directly via its URL — no login or download required.

How to use: prepend https://data.dataliteracy.cc to the file path shown below. For example:
https://data.dataliteracy.cc/simulated/practice_data.csv

Simulated datasets

File Description
simulated/practice_data.csv General practice dataset
simulated/polcomm_data.csv Political communication dataset
simulated/fake_demo_data.csv Fake demonstration dataset
simulated/robot_ancova_data.csv ANCOVA example dataset

Scripts

File Description
simulated/data_simulator.r R script used to simulate datasets
simulated/generate_demo_data.r R script to generate demonstration data

Adding or updating files

You do not need to clone the repository. Files can be added or changed directly via the GitHub website. Once merged, they are available here within minutes.
